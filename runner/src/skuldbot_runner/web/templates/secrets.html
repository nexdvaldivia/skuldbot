How to Use Secrets in Bots

In Bot DSL (Studio)

Reference secrets using the vault syntax:

${vault.my_secret_key}
${secret:api_credentials}

In Robot Framework

Secrets are injected as variables:

*** Variables ***
${API_KEY}    ${vault.api_key}

*** Test Cases ***
Use Secret
    Log    Using API key: ${API_KEY}

Security Note: Secrets are resolved at runtime on the Runner. They are never sent to the Orchestrator or stored in bot packages.
