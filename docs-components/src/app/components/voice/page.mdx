Voice automation with Azure Speech and Twilio. Text-to-speech, speech-to-text, phone calls, and voice-driven workflows. {{ className: 'lead' }}

<Note>
  Requires **Azure Speech Services** for TTS/STT and **Twilio** for phone calls. Store credentials in the Vault.
</Note>

<NodeHeader id="voice-speak" icon="Volume2" label="Speak" description="Convert text to speech" bgColor="#ccfbf1" textColor="#0f766e" />

<Properties>
  <Property name="type" type="string">
    `voice.speak`
  </Property>
</Properties>

### Configuration

<Properties>
  <Property name="text" type="textarea" required>
    **Text** — Text to convert to speech.
  </Property>
  <Property name="voice" type="select">
    **Voice** — Voice to use (e.g., `en-US-JennyNeural`).
  </Property>
  <Property name="output_path" type="text">
    **Output Path** — Save audio to file. Leave empty to play directly.
  </Property>
  <Property name="azure_speech_key" type="text" required>
    **Azure Key** — Use `${vault.AZURE_SPEECH_KEY}`.
  </Property>
  <Property name="azure_region" type="text">
    **Azure Region** — Azure region (e.g., `eastus`).
  </Property>
</Properties>

### Example

```json
{
  "text": "Hello ${customer.name}, your order has been shipped!",
  "voice": "en-US-JennyNeural",
  "output_path": "./audio/notification.mp3"
}
```

<NodeHeader id="voice-listen" icon="Mic" label="Listen" description="Convert speech to text" bgColor="#ccfbf1" textColor="#0f766e" />

<Properties>
  <Property name="type" type="string">
    `voice.listen`
  </Property>
</Properties>

### Configuration

<Properties>
  <Property name="audio_path" type="text" required>
    **Audio Path** — Path to audio file to transcribe.
  </Property>
  <Property name="language" type="text">
    **Language** — Language code (e.g., `en-US`, `es-ES`). Default: `en-US`.
  </Property>
  <Property name="azure_speech_key" type="text" required>
    **Azure Key** — Use `${vault.AZURE_SPEECH_KEY}`.
  </Property>
</Properties>

### Output

```json
{
  "text": "Hello, I'd like to check on my order status please.",
  "confidence": 0.95,
  "language": "en-US"
}
```

### Configuration

<Properties>
  <Property name="to" type="text" required>
    **To** — Phone number to call (E.164 format).
  </Property>
  <Property name="from" type="text" required>
    **From** — Your Twilio phone number.
  </Property>
  <Property name="message" type="textarea">
    **Message** — Text to speak when call connects.
  </Property>
  <Property name="twilio_sid" type="text" required>
    **Twilio SID** — Use `${vault.TWILIO_SID}`.
  </Property>
  <Property name="twilio_token" type="text" required>
    **Twilio Token** — Use `${vault.TWILIO_TOKEN}`.
  </Property>
