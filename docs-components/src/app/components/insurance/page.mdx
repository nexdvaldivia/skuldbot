<NodeHeader id="insurance-fnol_record" icon="FileText" label="Create FNOL Record" description="Create canonical First Notice of Loss record" bgColor="#fce7f3" textColor="#9d174d" />

<Properties>
  <Property name="type" type="string">
    `insurance.fnol_record`
  </Property>
</Properties>

**Configuration:**
- **caller_phone** (`text`, required) — Caller phone number
- **caller_name** (`text`) — Caller name
- **policy_number** (`text`) — Associated policy number
- **incident_type** (`select`, required) — Incident type:
  - `auto` — Auto/Vehicle
  - `property` — Property
  - `liability` — Liability
  - `health` — Health
  - `workers_comp` — Workers Compensation
  - `other` — Other
- **incident_date** (`text`) — Date of incident
- **incident_location** (`text`) — Location of incident
- **incident_description** (`textarea`, required) — Description of the incident
- **injuries** (`boolean`, default: `false`) — Injuries reported
- **police_report** (`text`) — Police report number
- **recording_url** (`text`) — Call recording URL
- **transcript** (`textarea`) — Call transcript

**Output:**
- **fnol_id** (`string`) — Unique FNOL record ID
- **fnol_record** (`object`) — Complete FNOL record object
- **created_at** (`string`) — Record creation timestamp

<NodeHeader id="insurance-lookup_policy" icon="Search" label="Lookup Policy" description="Find policy by phone, name, DOB (caller often doesn't have policy number)" bgColor="#fce7f3" textColor="#9d174d" />

<Properties>
  <Property name="type" type="string">
    `insurance.lookup_policy`
  </Property>
</Properties>

**Configuration:**
- **api_url** (`text`, required) — Policy Lookup API URL
- **phone_number** (`text`) — Phone number (from CallerID)
- **policyholder_name** (`text`) — Policyholder name
- **date_of_birth** (`text`) — Date of birth (YYYY-MM-DD)
- **ssn_last_four** (`text`) — SSN last 4 digits
- **address** (`text`) — Address
- **vin** (`text`) — Vehicle VIN (for auto)
- **policy_number** (`text`) — Policy number (if known)
- **method** (`select`, default: `POST`) — HTTP method: `GET`, `POST`
- **headers** (`textarea`) — Headers as JSON

**Output:**
- **found** (`boolean`) — Policy was found
- **policy_id** (`string`) — Policy internal ID
- **policy_number** (`string`) — Policy number
- **policyholder_name** (`string`) — Policyholder name
- **match_method** (`string`) — How policy was found (phone, name_dob, etc.)
- **raw_response** (`object`) — Raw API response

<NodeHeader id="insurance-validate_policy" icon="ShieldCheck" label="Validate Policy" description="Check if policy is active and has coverage for claim type" bgColor="#fce7f3" textColor="#9d174d" />

<Properties>
  <Property name="type" type="string">
    `insurance.validate_policy`
  </Property>
</Properties>

**Configuration:**
- **api_url** (`text`, required) — Policy Validation API URL
- **policy_id** (`text`) — Policy ID (from lookup)
- **policy_number** (`text`) — Policy number
- **claim_type** (`select`) — Claim type to validate:
  - `auto` — Auto/Vehicle
  - `property` — Property
  - `liability` — Liability
  - `health` — Health
  - `any` — Any Coverage
- **check_active** (`boolean`, default: `true`) — Check policy is active
- **check_coverage** (`boolean`, default: `true`) — Verify coverage type
- **method** (`select`, default: `POST`) — HTTP method: `GET`, `POST`
- **headers** (`textarea`) — Headers as JSON

**Output:**
- **valid** (`boolean`) — Policy is valid for this claim type
- **is_active** (`boolean`) — Policy is currently active
- **has_coverage** (`boolean`) — Policy has coverage for claim type
- **policyholder_name** (`string`) — Policyholder name
- **coverage_type** (`string`) — Type of coverage

<NodeHeader id="insurance-extract_claim_data" icon="FileSearch" label="Extract Claim Data" description="Use AI to extract structured claim data from conversation" bgColor="#fce7f3" textColor="#9d174d" />

<Properties>
  <Property name="type" type="string">
    `insurance.extract_claim_data`
  </Property>
</Properties>

**Configuration:**
- **transcript** (`textarea`, required) — Conversation transcript
- **claim_type** (`select`) — Expected claim type:
  - `auto` — Auto/Vehicle
  - `property` — Property
  - `any` — Any/Auto-detect
- **llm_provider** (`select`, default: `openai`) — LLM provider:
  - `openai` — OpenAI
  - `anthropic` — Anthropic Claude
  - `azure_openai` — Azure OpenAI

**Output:**
- **extracted_data** (`object`) — Structured claim data
- **confidence** (`number`) — Extraction confidence (0-1)
- **missing_fields** (`array`) — Fields that couldn't be extracted
