<NodeHeader id="document-ocr" icon="ScanLine" label="OCR Extract" description="Extract text with OCR" bgColor="#fef9c3" textColor="#854d0e" />

<Properties>
  <Property name="type" type="string">
    `document.ocr`
  </Property>
</Properties>

**Configuration:**
- **path** (`text`, required) — Image path
- **language** (`select`, default: `eng`) — OCR language:
  - `eng` — English
  - `spa` — Spanish
  - `deu` — German
  - `fra` — French

**Output Example:**

```json
{
  "text": "Extracted text from image...",
  "confidence": 0.92
}
```

<NodeHeader id="document-html_to_pdf" icon="FileDown" label="HTML to PDF" description="Convert HTML to PDF" bgColor="#fef9c3" textColor="#854d0e" />

<Properties>
  <Property name="type" type="string">
    `document.html_to_pdf`
  </Property>
</Properties>

**Configuration:**
- **html** (`textarea`, required) — HTML content or URL
- **output** (`text`, required) — Output PDF path

**Example:**

```json
{
  "type": "document.html_to_pdf",
  "config": {
    "html": "<html><body><h1>Invoice #${order.id}</h1><p>Total: $${order.total}</p></body></html>",
    "output": "./output/invoice_${order.id}.pdf"
  }
}
```

<NodeHeader id="document-pdf_fill_form" icon="PenTool" label="Fill PDF Form" description="Fill PDF form fields" bgColor="#fef9c3" textColor="#854d0e" />

<Properties>
  <Property name="type" type="string">
    `document.pdf_fill_form`
  </Property>
</Properties>

**Configuration:**
- **path** (`text`, required) — PDF form path
- **fields** (`textarea`, required) — Field values as JSON
- **output** (`text`, required) — Output path

**Example:**

```json
{
  "type": "document.pdf_fill_form",
  "config": {
    "path": "./forms/application.pdf",
    "fields": "{\"name\": \"${customer.name}\", \"date\": \"${today}\", \"amount\": \"${order.total}\"}",
    "output": "./output/filled_application.pdf"
  }
}
```
