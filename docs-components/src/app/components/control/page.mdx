### Configuration

<Properties>
  <Property name="items" type="text" required>
    Input Array
  </Property>
  <Property name="expression" type="text" required>
    Reducer Expression
  </Property>
  <Property name="initial" type="text">
    Initial Value (e.g., `0`) Default: `0`
  </Property>
  <Property name="item_var" type="text">
    Item Variable Name (e.g., `item`) Default: `item`
  </Property>
</Properties>

### Output Variables

Access these variables in subsequent nodes using `${Reduce/Aggregate.variableName}`.

<Properties>
  <Property name="result" type="any">
    Final accumulated value
  </Property>
  <Property name="count" type="number">
    Number of items processed
  </Property>
</Properties>
