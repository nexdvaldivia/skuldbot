Add human review to your automations. Approvals, reviews, and escalations for critical decisions. {{ className: 'lead' }}

<Note>
  Human tasks create notifications and wait for user action before continuing the workflow.
</Note>

---

<NodeHeader id="human-approval" icon="UserCheck" label="Request Approval" description="Wait for human approval" bgColor="#fef3c7" textColor="#92400e" />

<Properties>
  <Property name="type" type="string">
    `human.approval`
  </Property>
</Properties>

**Configuration:**
- **title** (`text`, required) — Request title
- **description** (`textarea`, required) — Details for the approver
- **approvers** (`text`) — Email addresses of approvers
- **timeout_minutes** (`number`, default: `60`) — Timeout in minutes

---

<NodeHeader id="human-input" icon="MessageSquare" label="Request Input" description="Request user input" bgColor="#fef3c7" textColor="#92400e" />

<Properties>
  <Property name="type" type="string">
    `human.input`
  </Property>
</Properties>

**Configuration:**
- **prompt** (`textarea`, required) — Question or instruction
- **input_type** (`select`, default: `text`) — Input type:
  - `text` — Text
  - `number` — Number
  - `choice` — Choice
  - `file` — File
- **options** (`text`) — Options if input type is Choice
