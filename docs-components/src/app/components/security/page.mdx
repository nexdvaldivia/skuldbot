<NodeHeader id="security-get_secret" icon="Key" label="Get Secret" description="Retrieve secret from vault" bgColor="#fee2e2" textColor="#991b1b" />

<Properties>
  <Property name="type" type="string">
    `security.get_secret`
  </Property>
</Properties>

**Configuration:**
- **name** (`text`, required) ‚Äî Secret name
- **vault** (`select`, default: `orchestrator`) ‚Äî Vault source:
  - `orchestrator` ‚Äî Orchestrator
  - `env` ‚Äî Environment
  - `azure` ‚Äî Azure Key Vault
  - `aws` ‚Äî AWS Secrets

**Using Vault Secrets:**

You can reference vault secrets directly in any node configuration:

```
${vault.API_KEY}
${vault.DB_PASSWORD}
${vault.OAUTH_CLIENT_SECRET}
```
