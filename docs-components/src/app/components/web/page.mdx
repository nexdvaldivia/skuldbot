  <path d="M2 12h20" />
</svg>
`} label="Open Browser" description="Open a web browser" bgColor="#dbeafe" textColor="#1d4ed8" />

<Properties>
  <Property name="type" type="string">
    `web.open_browser`
  </Property>
</Properties>

### When to Use

- Start a web scraping session
- Begin automated testing of web applications
- Log into web portals for data extraction
- Open multiple browser contexts for parallel processing

<Prerequisites>

- Browser driver installed (automatic with SkuldBot)
- Network access to target website

</Prerequisites>

### Configuration

<Properties>
  <Property name="url" type="text" required>
    URL (e.g., `https://example.com`)
  </Property>
  <Property name="browser" type="select">
    Browser Default: `chromium`
  </Property>
  <Property name="headless" type="boolean">
    Headless Mode (invisible browser) Default: `false`
  </Property>
</Properties>

### Setup Guide


<Collapsible title="Browser Options">

**Headless Mode:** Set `headless: true` to run without visible window (faster, ideal for servers)

**Proxy:** Configure `proxy.server`, `proxy.username`, and `proxy.password` using vault secrets

**Custom User Agent:** Set `userAgent` to a custom browser identification string

</Collapsible>


### Output Variables

Access these variables in subsequent nodes using `${Open Browser.variableName}`.

<Properties>
  <Property name="title" type="string">
    Page title
  </Property>
  <Property name="url" type="string">
    Current URL
  </Property>
</Properties>

### Example

```json
{
  "id": "node-1",
  "type": "web.open_browser",
  "label": "Open Browser",
  "config": {
    "url": "https://portal.example.com/login",
    "browser": "chromium",
    "headless": false
  },
  "outputs": {
    "success": "next-node",
    "error": "error-handler"
  }
}
```
