<NodeHeader id="ms365-connection" icon="Plug" label="MS365 Connection" description="Configure Microsoft 365 authentication" bgColor="#dbeafe" textColor="#1e40af" />

<Properties>
  <Property name="type" type="string">
    `ms365.connection`
  </Property>
</Properties>

**Configuration:**
- **tenant_id** (`text`, required) — Azure AD Tenant ID
- **client_id** (`text`, required) — Client (App) ID
- **client_secret** (`password`, required) — Client secret (use `${vault.ms365_secret}`)
- **user_email** (`text`) — Target mailbox email address

**Output:**
- **status** (`string`) — Connection status
- **authenticated** (`boolean`) — Whether authentication succeeded

## Email Read Operations

<NodeHeader id="ms365-email_list" icon="Mail" label="List Emails" description="List emails from a Microsoft 365 folder" bgColor="#dbeafe" textColor="#1e40af" />

<Properties>
  <Property name="type" type="string">
    `ms365.email_list`
  </Property>
</Properties>

**Configuration:**
- **folder** (`select`, default: `inbox`) — Mail folder: `inbox`, `sentitems`, `drafts`, `archive`, `deleteditems`, `junkemail`
- **top** (`number`, default: `10`) — Max results
- **filter** (`text`) — OData filter expression (e.g., `isRead eq false`)
- **search** (`text`) — Search query (e.g., `subject:invoice`)
- **order_by** (`select`, default: `receivedDateTime desc`) — Sort order

**Output:**
- **emails** (`array`) — List of emails with id, subject, from, to, bodyPreview, receivedDateTime, isRead, hasAttachments, importance
- **count** (`number`) — Number of emails returned

<NodeHeader id="ms365-email_get" icon="FileText" label="Get Email" description="Get a single email by ID with full details" bgColor="#dbeafe" textColor="#1e40af" />

<Properties>
  <Property name="type" type="string">
    `ms365.email_get`
  </Property>
</Properties>

**Configuration:**
- **message_id** (`text`, required) — Message ID
- **include_body** (`boolean`, default: `true`) — Include body
- **include_attachments** (`boolean`, default: `false`) — Include attachments

**Output:**
- **id** (`string`) — Email message ID
- **subject** (`string`) — Email subject
- **from** (`object`) — Sender info `{ name, email }`
- **to** (`array`) — To recipients
- **cc** (`array`) — CC recipients
- **bcc** (`array`) — BCC recipients
- **body** (`object`) — Email body `{ content, contentType }`
- **receivedDateTime** (`string`) — When received
- **hasAttachments** (`boolean`) — Has attachments
- **attachments** (`array`) — Attachment list if requested

## Required Graph API Permissions

| Operation | Permission |
|-----------|------------|
| Read emails | `Mail.Read` |
| Send emails | `Mail.Send` |
| Manage emails | `Mail.ReadWrite` |
| Calendar | `Calendars.ReadWrite` |
| OneDrive | `Files.ReadWrite` |
| Teams | `ChannelMessage.Send` |
