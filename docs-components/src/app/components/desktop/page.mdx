<NodeHeader id="desktop-open_app" icon="AppWindow" label="Open Application" description="Launch an application" bgColor="#f3e8ff" textColor="#7e22ce" />

<Properties>
  <Property name="type" type="string">
    `desktop.open_app`
  </Property>
</Properties>

**Configuration:**
- **path** (`text`, required) — Application path (e.g., `notepad`, `excel`, or full path)
- **args** (`text`) — Command-line arguments

**Example:**

```json
{
  "type": "desktop.open_app",
  "config": {
    "path": "C:\\Program Files\\Legacy App\\app.exe",
    "args": "--config production"
  }
}
```

<NodeHeader id="desktop-click" icon="MousePointer" label="Desktop Click" description="Click at coordinates or element" bgColor="#f3e8ff" textColor="#7e22ce" />

<Properties>
  <Property name="type" type="string">
    `desktop.click`
  </Property>
</Properties>

**Configuration:**
- **locator** (`text`) — Element locator (e.g., `name:Button1`)
- **x** (`number`) — X coordinate
- **y** (`number`) — Y coordinate

**Example - Click at coordinates:**

```json
{
  "type": "desktop.click",
  "config": {
    "x": 500,
    "y": 300
  }
}
```

**Example - Click by locator:**

```json
{
  "type": "desktop.click",
  "config": {
    "locator": "name:Save Button"
  }
}
```

<NodeHeader id="desktop-hotkey" icon="Command" label="Send Hotkey" description="Press keyboard shortcut" bgColor="#f3e8ff" textColor="#7e22ce" />

<Properties>
  <Property name="type" type="string">
    `desktop.hotkey`
  </Property>
</Properties>

**Configuration:**
- **keys** (`text`, required) — Key combination (e.g., `ctrl+c`, `alt+f4`, `ctrl+shift+s`)

**Common Hotkeys:**

| Hotkey | Action |
|--------|--------|
| `ctrl+s` | Save |
| `ctrl+c` | Copy |
| `ctrl+v` | Paste |
| `ctrl+z` | Undo |
| `ctrl+a` | Select All |
| `alt+f4` | Close Window |
| `enter` | Confirm |
| `escape` | Cancel |

<NodeHeader id="desktop-screenshot" icon="Monitor" label="Desktop Screenshot" description="Capture screen region" bgColor="#f3e8ff" textColor="#7e22ce" />

<Properties>
  <Property name="type" type="string">
    `desktop.screenshot`
  </Property>
</Properties>

**Configuration:**
- **path** (`text`, required) — Save path for screenshot
- **region** (`text`) — Region to capture (format: `x,y,width,height`). Leave empty for full screen.

**Example:**

```json
{
  "type": "desktop.screenshot",
  "config": {
    "path": "./evidence/screen.png",
    "region": "100,200,400,300"
  }
}
```

<NodeHeader id="desktop-image_click" icon="Image" label="Click Image" description="Find and click image on screen" bgColor="#f3e8ff" textColor="#7e22ce" />

<Properties>
  <Property name="type" type="string">
    `desktop.image_click`
  </Property>
</Properties>

**Configuration:**
- **image_path** (`text`, required) — Path to reference image
- **confidence** (`number`, default: `0.9`) — Match confidence (0-1)

**Example:**

```json

  "type": "desktop.image_click",
  "config": {
    "image_path": "./images/save_button.png",
    "confidence": 0.85
  }
}
```
