# Data Integration

Extract and load data from databases, files, and cloud services. {{ className: 'lead' }}

<Note>
  **23 components** for ETL pipelines. Taps extract data FROM sources, Targets load data TO destinations. All support expressions like `${vault.password}` for secure credentials.
</Note>

## Overview

| Type | Purpose | Count |
|------|---------|-------|
| **Taps** | Extract data FROM sources | 12 |
| **Targets** | Load data TO destinations | 11 |

<NodeHeader id="extract-sql-server" icon="Database" label="Extract SQL Server" description="Extract data from Microsoft SQL Server" bgColor="#dbeafe" textColor="#1e40af" />

<Properties>
  <Property name="type" type="string">
    `data.tap.sqlserver`
  </Property>
</Properties>

### Configuration

<Properties>
  <Property name="host" type="string" required>
    Database host. Supports `${vault.db_host}`.
  </Property>
  <Property name="port" type="number">
    Default: `1433`
  </Property>
  <Property name="database" type="string" required>
    Database name.
  </Property>
  <Property name="username" type="string" required>
    Username. Supports `${vault.db_user}`.
  </Property>
  <Property name="password" type="password" required>
    Password. **Use `${vault.db_password}`**.
  </Property>
  <Property name="query" type="textarea">
    SQL Query (e.g., `SELECT * FROM customers WHERE active = 1`).
  </Property>
  <Property name="table" type="string">
    Table name (if no query).
  </Property>
  <Property name="columns" type="string">
    Columns (comma-separated, e.g., `id, name, email`).
  </Property>
  <Property name="filter" type="string">
    WHERE clause filter (e.g., `status = 'active'`).
  </Property>
  <Property name="limit" type="number">
    Row limit.
  </Property>
  <Property name="mode" type="select">
    `memory` (< 50K rows), `batch` (large datasets), `stream` (direct pipe). Default: `memory`.
  </Property>
  <Property name="batch_size" type="number">
    Batch size. Default: `10000`.
  </Property>
</Properties>

### Output

<Properties>
  <Property name="records" type="array">
    Extracted records.
  </Property>
  <Property name="columns" type="array">
    Column names.
  </Property>
  <Property name="recordCount" type="number">
    Number of records.
  </Property>
</Properties>
