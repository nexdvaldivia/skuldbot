## Branching Logic

<NodeHeader id="control-if" icon="GitBranch" label="If Condition" description="Container: branch based on condition" bgColor="#e0f2fe" textColor="#0369a1" />

<Properties>
  <Property name="type" type="string">
    `control.if`
  </Property>
</Properties>

### When to Use

- Route orders based on total amount
- Skip processing for invalid records
- Different handling for VIP vs regular customers
- Conditional notifications

### Configuration

- **condition** (`text`, required) ‚Äî Condition to evaluate

### Output

- **result** (`boolean`) ‚Äî Condition evaluation result

### Condition Syntax

```javascript
// Comparisons
"${order.total} > 100"
"${status} == 'approved'"
"${items.length} >= 5"

// Logical operators
"${age} > 18 && ${country} == 'US'"
"${status} == 'pending' || ${priority} == 'high'"

// String methods
"${email}.includes('@company.com')"
"${name}.startsWith('Dr.')"

// Null checks
"${customer.address} != null"
"${phone} !== undefined"
```

## Loops

<NodeHeader id="control-loop" icon="Repeat" label="For Each" description="Container: iterate over a collection" bgColor="#e0f2fe" textColor="#0369a1" />

<Properties>
  <Property name="type" type="string">
    `control.loop`
  </Property>
</Properties>

### When to Use

- Process each row in a database query
- Send emails to a list of recipients
- Transform multiple files
- Batch API calls

### Configuration

- **items** (`text`, required) ‚Äî Items array (e.g., `${Query.records}`)
- **item_var** (`text`, default: `item`) ‚Äî Variable name for current item
- **accumulator_var** (`text`, default: `results`) ‚Äî Accumulator variable

### Output

- **index** (`number`) ‚Äî Current iteration index (0-based)
- **total** (`number`) ‚Äî Total number of items
- **isFirst** (`boolean`) ‚Äî True if first iteration
- **isLast** (`boolean`) ‚Äî True if last iteration

### When to Use

### Configuration

- **name** (`text`, required) ‚Äî Variable name

  "config": {

    "value": "${processedCount} + 1",
    "scope": "global"

## Data Transformation

<NodeHeader id="control-map" icon="Shuffle" label="Map/Transform" description="Transform each item in a collection" bgColor="#e0f2fe" textColor="#0369a1" />

<Properties>
  <Property name="type" type="string">
    `control.map`
  </Property>
</Properties>

### When to Use

- Rename fields in records
- Calculate derived values
- Format data for API calls
- Prepare data for reports

### Configuration

- **items** (`text`, required) ‚Äî Input array
- **expression** (`textarea`, required) ‚Äî Transform expression
- **item_var** (`text`, default: `item`) ‚Äî Item variable name

### Output

- **results** (`array`) ‚Äî Transformed array
- **count** (`number`) ‚Äî Number of items transformed

### Example: Format Names

```json
{
  "id": "map_1",
  "type": "control.map",
  "config": {
    "items": "${users}",
    "expression": "{ id: ${item.id}, fullName: ${item.first} + ' ' + ${item.last}, email: ${item.email}.toLowerCase() }",
    "item_var": "item"
  }
}
```

<NodeHeader id="control-filter" icon="Filter" label="Filter List" description="Filter items that match a condition" bgColor="#e0f2fe" textColor="#0369a1" />

<Properties>
  <Property name="type" type="string">
    `control.filter`
  </Property>
</Properties>

### When to Use

- Get only active records
- Exclude invalid data
- Segment lists
- Pre-process before loops

### Configuration

- **items** (`text`, required) ‚Äî Input array
- **condition** (`text`, required) ‚Äî Filter condition
- **item_var** (`text`, default: `item`) ‚Äî Item variable name

### Output

- **results** (`array`) ‚Äî Filtered array (matching items)
- **rejected** (`array`) ‚Äî Rejected array (non-matching items)
- **count** (`number`) ‚Äî Number of matching items

### Example: Active Users Only

```json
{
  "id": "filter_1",
  "type": "control.filter",
  "config": {
    "items": "${users}",
    "condition": "${item.status} == 'active' && ${item.email} != null",
    "item_var": "item"
  }
}
```

<NodeHeader id="control-reduce" icon="Combine" label="Reduce/Aggregate" description="Reduce array to a single value" bgColor="#e0f2fe" textColor="#0369a1" />

<Properties>
  <Property name="type" type="string">
    `control.reduce`
  </Property>
</Properties>

### When to Use

- Calculate totals
- Count items by criteria
- Find min/max values
- Aggregate statistics

### Configuration

- **items** (`text`, required) ‚Äî Input array
- **expression** (`textarea`, required) ‚Äî Reduce expression
- **initial** (`text`, default: `0`) ‚Äî Initial accumulator value
- **item_var** (`text`, default: `item`) ‚Äî Item variable name
- **acc_var** (`text`, default: `acc`) ‚Äî Accumulator variable name

### Output

- **result** (`any`) ‚Äî Final reduced value
- **count** (`number`) ‚Äî Number of items processed

### Example: Sum Order Totals

```json
{
  "id": "reduce_1",
  "type": "control.reduce",
  "config": {
    "items": "${orders}",
    "expression": "${acc} + ${item.total}",
    "initial": "0",
    "item_var": "item",
    "acc_var": "acc"
  }
}
```

**Output:** `{ "result": 15750.50, "count": 45 }`
