# Excel & CSV

Complete Excel automation. Read, write, format cells, create charts, pivot tables, and work with CSV files. {{ className: 'lead' }}

<Note>
  **27 components**. Uses **openpyxl** for Excel files and native Python for CSV. Works without Microsoft Excel installed. Connect a [Storage Provider](/components/storage) to read/write Excel files from S3, Azure Blob, SharePoint, or any cloud storage.
</Note>

### Storage Provider Integration

All Excel nodes that read or write files (`open`, `save`, `create`, `csv_read`, `csv_write`, `export_pdf`) support the **Storage Provider** connection. Connect a Storage Provider node to your Excel nodes to access files from any backend:

```
[Storage Provider (S3)]
       │ storage handle
       ▼
[Excel Open] → path: "reports/monthly.xlsx"
       │         (resolved via S3, downloaded to temp, opened locally)
       ▼
[Excel Read Range] → reads data
       ▼
[Excel Save] → saves and uploads back to S3
```

Without a Storage Provider connected, Excel nodes use the local filesystem (Studio debug mode).

## Overview

| Node | Purpose |
|------|---------|
| **Read Range** | Read a block of cells |
| **Write Range** | Write data to cells |
| **Add Row** | Append data to sheet |
| **Read/Write Cell** | Single cell operations |
| **CSV Read/Write** | Work with CSV files |

### How It Works

Reads a range of cells and returns the data as a 2D array or array of objects (if headers are present).

### Configuration

<Properties>
  <Property name="range" type="text" required>
    **Range** — Cell range (e.g., `A1:D100`, `A:D` for full columns).
  </Property>
  <Property name="sheet" type="text">
    **Sheet** — Sheet name. Default: active sheet.
  </Property>
  <Property name="headers" type="boolean">
    **Has Headers** — First row contains column names. Default: true.
  </Property>
</Properties>

### Output

With `headers: true`:
```json
{
  "data": [
    {"Name": "John", "Email": "john@example.com", "Amount": 100},
    {"Name": "Jane", "Email": "jane@example.com", "Amount": 200}
  ],
  "rows": 2,
  "columns": 3
}
```

With `headers: false`:
```json
{
  "data": [
    ["Name", "Email", "Amount"],
    ["John", "john@example.com", 100],
    ["Jane", "jane@example.com", 200]
  ],
  "rows": 3,
  "columns": 3
}
```

### Set Format

<Properties>
  <Property name="type" type="string">
    `excel.set_format`
  </Property>
</Properties>

**Configuration:**
- **range** — Cell range to format.
- **bold** — Make text bold.
- **font_color** — Text color (hex).
- **bg_color** — Background color (hex).
- **number_format** — Number format (e.g., `#,##0.00`, `0%`).

### Example: Format Header Row

```json
{
  "range": "A1:E1",
  "bold": true,
  "bg_color": "#4472C4",
  "font_color": "#FFFFFF"
}
```

### Merge Cells

```json
{
  "type": "excel.merge_cells",
  "config": {
    "range": "A1:D1"
  }
}
```
